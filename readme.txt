
1. Для перевірки коду вводимо повне посилання  (наприклад, https://www.youtube.com/watch?v=T_L8_pSme9o)

2. Для перевірки  "проваленого запиту" (відсутній інтернет, сервер відповів відмовою, тощо) - 
       так само вводимо повне посилання (при відключеному Wi-Fi, наприклад)

3. Запит по валідному посиланню відбувається кожні 10 сек. Кількість запитів - у віконці "Request count"

4. При невалідному посиланні код зафіксує помилку.
